# Air Quality Sensor

This is a project by Jay Herron that uses the si7021 to read temp and humidity

## Wiring

See each module's driver documentation for more information

### SI7021

* Vin <-> D4
* GND <-> D2
* SCL <-> D1
* SDA <-> D0

Configuring it this way allows me to solder the sensor chip directly onto the particle headers.
